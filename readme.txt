A modern and responsive WooCommerce theme for electronic stores.

== Description ==

Remote Store is a modern, responsive WordPress theme designed specifically for WooCommerce stores selling electronic products. The theme offers a clean, user-friendly interface with a focus on product showcasing and easy navigation. It includes customizable homepage sections, dynamic product displays, and full integration with WooCommerce.

= Features =
* Fully responsive design
* WooCommerce integration
* Customizable homepage sections
* Product category showcase
* Featured product sections
* Dynamic product grids
* Clean and modern design
* SEO optimized
* Cross-browser compatibility
* Translation ready

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
4. Install and activate WooCommerce plugin.
5. Create products and categories.
6. Customize the theme settings from Appearance > Customize.

== Frequently Asked Questions ==

= Does this theme require any plugins? =

This theme requires WooCommerce plugin to be installed and activated.

= How do I set up the homepage? =

1. Create a static page for your homepage.
2. Go to Settings > Reading and set your static page as the homepage.
3. Go to Appearance > Customize > Homepage Sections to configure the homepage sections.
4. Select the categories and products you want to display on the homepage.

= Can I customize the colors and fonts? =

Yes, you can customize the colors, fonts, and other theme options from Appearance > Customize.

= Is this theme compatible with page builders? =

Yes, this theme is compatible with most page builders, including Elementor, Beaver Builder, and Divi.
